│                                                                  │

┌─ ШАГ 2: ПРОВЕРИТЬ ЛОКАЛЬНО ───────────────────────────────────┐
│                                                                  │
│  Команда:                                                        │
│  > npm run dev                                                   │
│                                                                  │
│  Откройте в браузере:                                            │
│  http://localhost:3000/admin/auto-payments                       │
│                                                                  │
│  Нажмите кнопку "Запустить проверку"                             │
│                                                                  │
└──────────────────────────────────────────────────────────────────┘

┌─ ШАГ 3: НАСТРОИТЬ КРЕДИТЫ ────────────────────────────────────┐
│                                                                  │
│  1. Откройте: http://localhost:3000/loans                        │
│  2. Нажмите "Изменить" на любом кредите                          │
│  3. Заполните поле "Дата следующего платежа"                     │
│     (например: 15.10.2025)                                       │
│  4. Сохраните                                                    │
│                                                                  │
│  ✅ Готово! Система начнёт отслеживать этот кредит              │
│                                                                  │
└──────────────────────────────────────────────────────────────────┘

┌─ КАК ЭТО РАБОТАЕТ ────────────────────────────────────────────┐
│                                                                  │
│  • Каждый день в 12:00 МСК система проверяет кредиты            │
│  • Если до платежа осталось ≤ 10 дней → создаётся напоминание   │
│  • Напоминание появляется на странице "Платежи"                  │
│  • Вы никогда не пропустите платёж! 🎉                          │
│                                                                  │
