# Редизайн карточки тендера

**Дата:** 13.11.2025  
**Статус:** Завершено

## Обзор

Выполнен полный редизайн карточек тендеров на странице `/tenders/department` с улучшенным визуальным оформлением и добавлением функции копирования номера ЕИС.

## Основные улучшения

### 1. Визуальные улучшения

#### Карточка тендера
- Увеличен border-radius до 0.75rem
- Улучшена тень: многослойная с плавным переходом
- Добавлена верхняя цветная полоса (градиент), появляющаяся при наведении
- Плавная анимация при наведении: translateY(-4px)
- Увеличен padding до 1rem
- Анимация при перетаскивании: rotate(2deg)

#### Номер ЕИС
- Новый контейнер с градиентным фоном (голубой)
- Кнопка копирования с иконкой SVG
- Моноширинный шрифт для номера
- Анимация кнопки при наведении

#### Финансовая информация
- Зелёный градиентный фон
- Моноширинный шрифт для сумм
- Улучшенная контрастность

### 2. Функция копирования номера ЕИС

Добавлена функция copyToClipboard с использованием Clipboard API.
Кнопка с иконкой копирования и анимацией.

## Изменённые файлы

1. components/tenders/tender-kanban.tsx
2. components/tenders/tender-kanban.module.css

## Результат

Карточки тендеров теперь имеют современный дизайн с градиентами, анимациями и улучшенной типографикой.
